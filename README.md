<h1 align="center">T1T5</h1>



#### Setup 

```
 pip3 install -r requirements.txt
 python3 t1t5.py
 ```

#### How to run 

1.Set a target 
```
>T1T5 >> set example.com
```
2.Run options 
```
>T1T5 >> 1             --> Single runs 
>T1T5 >> 1,2,3,4       --> Multiple Runs 
```
###### For any help or to see avaliable options type : ``` help ```
